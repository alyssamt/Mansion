MECHANICS:
1. Win: All kids are scared away; Lose: Any kid reaches objective on third floor
2. Set traps before level starts
3. Move around and scare during level
4. Kids have a "scare meter" which will cause them to run away if filled
5. Vamp can approach kid from behind to suck their blood, filling the kid's scare meter and giving Vamp an energy boost
